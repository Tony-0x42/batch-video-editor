# AI 去水印

## 1. 页面概述

- **页面定位**：底部导航「AI 去水印」入口，核心功能为短视频平台水印去除。
- **用户角色**：所有已登录用户。
- **入口来源**：底部导航栏「AI 去水印」。
- **视觉规范**：页面颜色为蓝色，跟前页面颜色保持一致。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：无（一级页面）。
- **页面标题**：「AI 去水印」。

### 2.2 链接输入区

- **元素**：单行文本输入框。
- **占位文案**：「请粘贴短视频平台分享链接」。
- **支持平台**：抖音、小红书等主流短视频平台。
- **操作按钮**：
  - 【解析】蓝色按钮，点击后根据链接下载对应原视频文件并自动去除画面水印区域
  - 【清空】一键清空输入框全部链接文本

### 2.3 解析结果展示区

- **元素**：解析完成后分页签展示资源。
- **页签包括**：
  - **视频标签**：去水印后的成片视频，支持在线预览
  - **图片标签**：视频内截取帧图片
  - **文本标签**：视频附带文案、标题

### 2.4 保存功能

- **元素**：蓝色「保存」按钮。
- **功能**：点击将去水印后的视频 / 图片保存至手机本地相册。
- **状态**：未解析完成前按钮隐藏或置灰。

### 2.5 历史记录（可选）

- **元素**：展示用户近期的解析记录列表。
- **功能**：点击可重新查看解析结果。

## 3. 交互流程

1. 用户点击底部导航「AI 去水印」进入本页面。
2. 用户在输入框粘贴短视频平台分享链接。
3. 用户点击【解析】按钮，前端校验链接是否为空。
4. 校验通过后调用解析接口，展示加载动画。
5. 解析完成后分页签展示视频、图片、文本结果。
6. 用户预览视频/图片，点击【保存】按钮下载到本地相册。
7. 用户点击【清空】可清空输入框重新输入链接。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 分享链接 | String | 用户输入 | 抖音、小红书等平台分享链接 |
| 去水印视频 URL | URL | 接口返回 | 处理后的成片视频 |
| 视频帧图片列表 | Array<URL> | 接口返回 | 视频内截取帧图片 |
| 视频文案 | String | 接口返回 | 视频附带文案、标题 |
| 解析状态 | Enum | 接口返回 | 待解析、解析中、已完成、失败 |

## 5. 异常处理与边界规则

- **链接为空**：点击解析时提示「请输入分享链接」。
- **链接格式错误**：提示「链接格式不正确，请检查链接」。
- **链接解析失败**：弹窗提示「链接解析失败，请检查链接有效性或更换网络重试」。
- **保存失败**：提示「保存失败，请检查存储权限」。
- **解析平台不支持**：提示「暂不支持该平台链接解析」。
- **网络异常**：提示「网络连接失败，请检查网络后重试」。

## 6. UI 与视觉规范

- 一级页面，底部有固定导航栏。
- 输入框区域居中或居上，占位文案灰色。
- 【解析】按钮蓝色填充主按钮；【清空】按钮灰色/白色次要按钮。
- 解析结果页签选中项蓝色下划线/文字，未选中项灰色。
- 视频预览区默认居中，下方展示【保存】按钮。
- 解析过程中展示加载动画。
