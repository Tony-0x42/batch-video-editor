# 资料查看页（学习专区）

## 1. 页面概述

- **页面定位**：首页「学习专区」入口的二级页面，用于集中展示后台上传的教学视频、知识文档等学习资料。
- **用户角色**：所有已登录用户。
- **入口来源**：首页 → 学习专区。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回首页。
- **页面标题**：「学习专区」或「资料查看」。

### 2.2 资料分类 Tab

- **元素**：顶部横向 Tab，如「全部」「视频」「文档」。
- **功能**：切换筛选条件，列表根据分类展示对应资料。
- **数据约束**：分类名称与内容由后台配置。

### 2.3 资料列表

- **元素**：列表或卡片形式展示每条资料。
- **每条资料展示字段**：
  - 封面图/缩略图
  - 资料标题
  - 资料类型标签（视频/文档/PDF）
  - 发布时间
  - 浏览次数（可选）
- **操作**：点击整条资料进入资料详情页/播放页。

### 2.4 搜索框（可选）

- **元素**：顶部搜索输入框。
- **功能**：输入关键词后模糊搜索资料标题。
- **清空按钮**：一键清空搜索关键词。

### 2.5 空状态

- **触发条件**：后台未配置任何资料或搜索结果为空。
- **展示**：空页面占位图 + 文案「暂无学习资料」。

## 3. 交互流程

1. 用户从首页点击「学习专区」进入本页面。
2. 页面自动拉取资料列表，默认展示「全部」分类。
3. 用户切换 Tab，列表重新筛选展示。
4. 用户点击某条资料，进入详情页查看文档或播放视频。
5. 用户输入关键词搜索，列表展示匹配结果。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 分类列表 | Array<String> | 后台配置 | 如「全部、视频、文档」 |
| 资料列表 | Array<Object> | 后台配置/接口 | 每条包含：id、标题、类型、封面 URL、内容 URL、发布时间、浏览次数 |
| 搜索关键词 | String | 用户输入 | 本地过滤或接口查询 |

## 5. 异常处理与边界规则

- **后台无资料**：展示空状态占位图。
- **搜索无结果**：提示「未找到相关资料」。
- **视频/文档加载失败**：提示「内容加载失败，请重试」。
- **网络异常**：下拉刷新失败时提示网络错误。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题，符合二级页面统一规范。
- Tab 选中项蓝色下划线，未选中项灰色文字。
- 资料卡片封面图圆角，标题黑色，类型标签蓝色/灰色。
- 空状态居中展示，插画 + 灰色文案。
