# 新手文档页（教程 & 文档）

## 1. 页面概述

- **页面定位**：首页「教程 & 文档」入口的二级页面，集中展示新手快速上手、操作指南、常见问题 FAQ 等文档内容。
- **用户角色**：所有已登录用户。
- **入口来源**：首页 → 教程 & 文档入口。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回首页。
- **页面标题**：「新手指南」或「帮助中心」。

### 2.2 文档分类 Tab

- **元素**：顶部横向 Tab，如「快速上手」「操作指南」「常见问题」。
- **功能**：切换分类后展示对应文档列表。
- **数据约束**：分类名称由后台配置。

### 2.3 文档列表

- **元素**：列表展示文档标题。
- **每条展示字段**：
  - 文档标题
  - 更新时间
- **操作**：点击标题展开/进入文档详情。

### 2.4 文档详情

- **元素**：富文本/图文混排展示文档内容。
- **内容类型**：支持文字、图片、步骤说明。
- **操作**：支持上下滑动查看全文；图片支持放大预览。

### 2.5 搜索框（可选）

- **元素**：顶部搜索框，可输入关键词搜索文档标题或内容。

## 3. 交互流程

1. 用户从首页点击教程入口进入本页面。
2. 页面拉取文档分类与列表。
3. 用户切换 Tab 浏览不同分类文档。
4. 用户点击文档标题，进入详情页查看完整内容。
5. 用户可使用搜索框查找特定问题。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 文档分类 | Array<String> | 后台配置 | 如「快速上手、操作指南、常见问题」 |
| 文档列表 | Array<Object> | 后台配置 | 每条包含：id、标题、分类、更新时间、内容 |
| 文档详情 | Object | 后台配置 | 富文本内容 |

## 5. 异常处理与边界规则

- **后台无文档**：展示空状态占位图。
- **文档内容加载失败**：提示「文档加载失败，请重试」。
- **搜索无结果**：提示「未找到相关文档」。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题。
- Tab 选中项蓝色下划线。
- 文档列表项标题黑色，更新时间灰色小字。
- 详情页正文黑色，图片圆角，步骤标题蓝色加粗。
