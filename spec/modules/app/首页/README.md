# 首页

## 1. 页面概述

- **页面定位**：APP 落地首屏，核心承载品牌宣传、业绩公示、功能导航、学习资料、合作品牌展示。
- **用户角色**：所有已登录用户（分公司、服务商、个人账号）。
- **入口来源**：底部导航栏「首页」入口；APP 登录成功后默认进入。

## 2. 页面元素与功能点

### 2.1 顶部后台贴图展示区（轮播 / 海报区）

- **元素**：轮播图组件，支持多张图片横向滑动或自动轮播。
- **功能**：
  - 展示会展海报、品牌宣发图、活动物料等图片素材。
  - 支持手动左右滑动切换。
  - 支持自动轮播，轮播间隔由后台配置（默认 3 秒）。
  - 图片点击可跳转指定链接或海报详情页（若后台配置了跳转地址）。
- **数据约束**：所有图片由后台上传、新增、替换、删除；前端无本地上传/编辑权限，仅渲染后台下发素材。

### 2.2 喜报数据栏

- **元素**：业绩标题 + 销售冠军数据卡片。
- **展示内容**：
  - 业绩标题：如「恭喜华东区超额完成 Q2 目标！」
  - 销售冠军数据：姓名 + 销售金额，如「王小明 - ¥586,000」
- **数据约束**：所有文字、金额、区域名称、人员姓名均由后台编辑更新，前端实时拉取最新数据展示。

### 2.3 功能导航入口区

以宫格或卡片形式展示以下入口，点击后跳转对应二级页面：

| 入口 | 跳转目标 | 功能说明 |
|---|---|---|
| 学习专区 | 资料查看页 | 展示后台上传的教学视频、知识文档，支持在线查看 |
| 信息咨询 | 联系方式页 | 展示在线客服、总台、各地经销负责人电话 |
| 品牌专区 | 品牌专区页 | 展示合作品牌 LOGO、品牌介绍、品牌图片 |
| 其他服务 | 预留空白 | 当前状态：预留空白区块，暂不开发功能，后续迭代扩展 |

### 2.4 教程 & 文档入口

- **元素**：卡片或横幅入口，文案如「新手指南」「快速上手」。
- **跳转目标**：新手文档页。
- **内容包含**：快速上手、操作指南、常见问题 FAQ。
- **数据约束**：后台上传更新全套图文教程。

### 2.5 底部固定导航栏

- **元素**：首页、AI 云创、AI 去水印、我的。
- **规则**：全 APP 一级页面底部固定 4 项导航入口，顺序固定；当前选中项高亮显示。

## 3. 交互流程

1. 用户进入首页，页面自动拉取后台配置数据（轮播图、喜报、功能入口、教程入口）。
2. 轮播图自动播放，用户可手动滑动切换。
3. 用户点击功能导航入口，跳转对应二级页面。
4. 用户点击教程入口，跳转新手文档页。
5. 用户点击底部导航切换其他一级页面。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 轮播图列表 | Array<Object> | 后台配置 | 每张图片包含：id、图片 URL、跳转链接、排序权重 |
| 业绩标题 | String | 后台配置 | 喜报标题 |
| 销售冠军姓名 | String | 后台配置 | 如「王小明」 |
| 销售金额 | String/Number | 后台配置 | 如「586000」，前端格式化为「¥586,000」 |
| 功能入口配置 | Array<Object> | 后台配置 | 入口名称、图标、跳转目标、是否启用 |
| 教程入口配置 | Object | 后台配置 | 标题、封面图、文档内容/链接 |

## 5. 异常处理与边界规则

- **后台无轮播图配置**：展示默认空白占位图，不报错崩溃。
- **后台无喜报数据**：隐藏喜报栏或展示默认空状态。
- **功能入口未配置**：展示默认 4 个入口，点击后若未配置内容则提示「功能开发中」。
- **网络异常**：页面下拉刷新失败时提示「网络连接失败，请下拉重试」。
- **图片加载失败**：展示默认占位图，支持重新加载。

## 6. UI 与视觉规范

- 页面顶部无返回按钮（一级页面）。
- 轮播图指示器：小圆点样式，当前选中项蓝色，未选中项灰色。
- 喜报栏：醒目背景色（如浅金色/浅蓝色），关键数字蓝色高亮。
- 功能导航入口：图标 + 文字，宫格布局，图标风格统一。
- 教程入口：卡片样式，封面图 + 标题。
- 底部导航栏：白色背景，图标 + 文字，选中项蓝色，未选中项灰色。
