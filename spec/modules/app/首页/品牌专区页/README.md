# 品牌专区页

## 1. 页面概述

- **页面定位**：首页「品牌专区」入口的二级页面，用于展示合作品牌 LOGO、品牌介绍、品牌图片等。
- **用户角色**：所有已登录用户。
- **入口来源**：首页 → 品牌专区。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回首页。
- **页面标题**：「品牌专区」。

### 2.2 品牌列表

- **元素**：卡片或宫格列表展示合作品牌。
- **每条品牌展示字段**：
  - 品牌 LOGO
  - 品牌名称
  - 品牌简介（一行或两行省略）
- **操作**：点击品牌卡片进入品牌详情页。

### 2.3 品牌详情页

- **元素**：品牌详情页展示品牌的完整信息。
- **展示字段**：
  - 品牌 LOGO
  - 品牌名称
  - 品牌详细介绍
  - 品牌宣传图片/视频
- **操作**：图片支持放大预览，视频支持播放。

### 2.4 空状态

- **触发条件**：后台未配置任何品牌信息。
- **展示**：空页面占位图 + 文案「暂无品牌信息」。

## 3. 交互流程

1. 用户从首页点击「品牌专区」进入本页面。
2. 页面拉取品牌列表数据。
3. 用户点击品牌卡片，进入品牌详情页。
4. 用户在详情页查看品牌介绍、图片、视频。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 品牌列表 | Array<Object> | 后台配置 | 每条包含：id、LOGO、名称、简介、排序权重 |
| 品牌详情 | Object | 后台配置 | 详情包含：id、名称、LOGO、详细介绍、图片/视频列表 |

## 5. 异常处理与边界规则

- **后台无品牌数据**：展示空状态占位图。
- **图片加载失败**：展示默认占位图。
- **视频播放失败**：提示「视频播放失败，请重试」。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题。
- 品牌卡片白色背景、圆角、LOGO 居中。
- 品牌名称黑色加粗，简介灰色小字。
- 详情页图片可横向滑动查看。
