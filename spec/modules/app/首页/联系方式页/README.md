# 联系方式页（信息咨询）

## 1. 页面概述

- **页面定位**：首页「信息咨询」入口的二级页面，用于展示平台客服、总台及各地经销负责人的联系方式。
- **用户角色**：所有已登录用户。
- **入口来源**：首页 → 信息咨询。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回首页。
- **页面标题**：「信息咨询」或「联系方式」。

### 2.2 在线客服入口

- **元素**：卡片形式展示在线客服入口。
- **功能**：点击后跳转在线客服聊天窗口或唤起客服电话/微信客服。
- **数据约束**：客服链接/联系方式由后台配置。

### 2.3 总台联系方式

- **元素**：展示总台电话、服务时间。
- **功能**：点击电话号码可直接唤起系统拨号。
- **展示字段**：
  - 总台电话
  - 服务时段（如 早 9:00-18:00）

### 2.4 各地经销负责人列表

- **元素**：列表展示各区域经销负责人信息。
- **每条展示字段**：
  - 负责人姓名
  - 负责区域
  - 联系电话
- **操作**：点击电话号码唤起系统拨号。

### 2.5 空状态

- **触发条件**：后台未配置任何联系方式。
- **展示**：提示「暂无联系方式，请稍后查看」。

## 3. 交互流程

1. 用户从首页点击「信息咨询」进入本页面。
2. 页面拉取后台配置的联系方式数据。
3. 用户点击在线客服入口，跳转客服页面。
4. 用户点击电话，唤起拨号弹窗。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 在线客服入口 | Object | 后台配置 | 入口名称、图标、跳转链接/客服号 |
| 总台电话 | String | 后台配置 | 可点击拨号 |
| 服务时段 | String | 后台配置 | 如「早 9:00-18:00」 |
| 负责人列表 | Array<Object> | 后台配置 | 姓名、区域、电话 |

## 5. 异常处理与边界规则

- **后台无配置**：展示空状态，不报错。
- **电话拨打失败**：提示「拨号失败，请检查设备权限」。
- **客服链接失效**：提示「客服入口暂不可用」。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题。
- 联系信息卡片白色背景、圆角、轻微阴影。
- 电话文字蓝色可点击，其他文字灰色/黑色。
- 列表项之间保持统一间距。
