# AI 云创（视频素材库列表页）

## 1. 页面概述

- **页面定位**：底部导航「AI 云创」入口，展示用户创建的所有视频素材组列表，是 AI 视频创作的主入口。
- **用户角色**：所有已登录用户。
- **入口来源**：底部导航栏「AI 云创」。
- **核心前置规则**：算力不足时，禁用视频生成、视频下载功能。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：无（一级页面）。
- **页面标题**：「AI 云创」。
- **搜索框**：顶部搜索框，输入视频组名称模糊检索对应视频组。

### 2.2 算力统计区块

- **元素**：卡片展示算力消耗情况。
- **展示字段**：
  - 算力消耗百分比（如 35.6%）
  - 已消耗算力 / 总算力（如 356/1000 GF）
- **数据规则**：
  - 账号总算力额度由后台分配、修改。
  - 每生成一条视频消耗对应算力，消耗数值与生成视频数量联动。
  - 当「已消耗算力 = 总算力」时，锁定全部视频生成、下载按钮，弹窗提示用户算力不足。

### 2.3 视频素材库列表

- **元素**：列表展示每条视频组。
- **每条视频组固定展示字段**：
  - 视频组名称
  - 创建日期（格式 YYYY-MM-DD）
  - 已生成数量 / 本组最大生成上限（例：已生成 3/10 条，颜色统一为灰色）
- **操作按钮**：
  - 展开详情：点击视频组进入视频组编辑页
  - 删除：单条视频组删除按钮，二次确认弹窗后删除本组全部素材与生成记录

### 2.4 列表操作功能

- **搜索**：顶部搜索框，输入视频名称模糊检索对应视频组。
- **排序**：长按视频组条目，支持上下拖拽自定义排序。
- **删除**：单条视频组删除按钮，二次确认弹窗后删除本组全部素材与生成记录。
- **新增视频素材**：蓝色按钮，点击创建全新空白视频组。

### 2.5 新增视频素材入口

- **顶部「+」按钮**：进入视频组创建流程。
- **底部「新增视频素材」按钮**：蓝色主按钮，同样进入创建流程。

### 2.6 空状态

- **触发条件**：用户尚未创建任何视频组。
- **展示**：空白提示文案 +「新增视频素材」入口。

## 3. 交互流程

1. 用户点击底部导航「AI 云创」进入本页面。
2. 页面拉取算力数据和视频组列表。
3. 用户可搜索、拖拽排序、删除视频组。
4. 用户点击视频组进入视频组编辑页。
5. 用户点击「新增视频素材」创建新的空白视频组。
6. 若算力耗尽，新增/生成按钮置灰，点击时弹窗提示。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 总算力 | Number | 后台配置/账号配置 | 单位 GF |
| 已消耗算力 | Number | 根据生成记录计算 | 单位 GF |
| 算力消耗百分比 | Number | 前端计算 | 已消耗/总算力 * 100% |
| 视频组列表 | Array<Object> | 后台/本地 | id、名称、创建日期、已生成数量、最大生成上限 |
| 搜索关键词 | String | 用户输入 | 本地或接口模糊查询 |

## 5. 异常处理与边界规则

- **算力耗尽**：生成、下载按钮置灰，弹窗提示「当前算力已耗尽，请联系管理员增加算力额度」。
- **搜索无结果**：提示「未找到相关视频组」。
- **删除视频组**：二次确认弹窗，确认后删除本组全部素材与生成记录。
- **网络异常**：下拉刷新失败时提示网络错误。

## 6. UI 与视觉规范

- 一级页面，无返回按钮，底部有固定导航栏。
- 算力统计卡片蓝色主题，百分比数字蓝色高亮。
- 视频组列表项白色卡片、圆角、阴影。
- 已生成数量文字统一为灰色。
- 新增按钮为蓝色填充主按钮。
- 删除按钮为红色或带删除图标，点击后二次确认。
