# 视频组编辑页

## 1. 页面概述

- **页面定位**：AI 云创核心操作页，用户在此上传视频素材、进行 AI 分割、管理分镜头、配音、添加背景音乐并批量生成成片。
- **用户角色**：所有已登录用户。
- **入口来源**：AI 云创素材库列表页 → 点击任意视频组进入。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回 AI 云创素材库列表页。
- **页面标题**：视频组名称，可编辑（点击标题进入编辑状态）。

### 2.2 视频预览区

- **元素**：顶部视频预览窗口。
- **视觉规范**：视频名称上下添加虚化遮罩。
- **功能**：播放当前选中的分镜头视频或整体预览。

### 2.3 分镜头总进度展示

- **元素**：进度文字。
- **格式**：「分镜头 X/Y 已生成 A/B」，例：「分镜头 2/50 已生成 0/0」。
- **说明**：X 为当前分镜头序号，Y 为总分镜头数，A 为已生成成片数，B 为待生成数。

### 2.4 音频基础选项

- **元素**：保留原视频原声开关。
- **功能**：开启后生成的成片保留原视频声音；关闭则不保留。

### 2.5 顶部全局操作

- **AI 视频分割按钮**：进入视频分割流程。
- **查看切割进度按钮**：进入视频分割进度页。
- **视觉规范**：字幕为蓝色，底色为淡蓝色，色彩和前页面保持一致。

### 2.6 分镜头素材管理区

- **分镜头条目**：每条分镜头带编号（1、2…）。
- **单镜头操作**：
  - 移动排序：长按拖拽调整分镜头顺序
  - 镜像翻转：对当前分镜头视频进行水平镜像
  - 上传独立视频段：替换当前分镜头素材
- **素材调整**：分镜头内视频素材支持上下、左右拖拽调整顺序。
- **预览功能**：点击任意分镜头视频，弹出全屏播放预览窗口。
- **删除**：单镜头右上角关闭按钮，可删除当前分镜头素材。

### 2.7 AI 口播配音模块

- **两种配音方式**：
  - 文字转语音：输入文本，AI 生成配音音频
  - 本地录音：手机麦克风实时录音生成配音
- **功能**：音频可叠加至分镜头视频轨道。
- **展示字段**：配音文本、配音语速/音色选择（如后台支持）。

### 2.8 背景音乐模块

- **两种音频来源二选一 / 叠加**：
  - 本地上传背景音乐文件
  - 提取原视频自带音频作为背景音乐
- **功能**：选择/上传背景音乐后，可调整音量、裁剪片段。

### 2.9 AI 生成按钮

- **元素**：页面底部蓝色主按钮。
- **文案**：「AI 生成」。
- **功能**：点击消耗算力批量生成分镜头成片。
- **状态**：算力耗尽时按钮置灰不可点击。

## 3. 交互流程

1. 用户从素材库点击视频组进入编辑页。
2. 页面加载视频组信息、分镜头素材、算力数据。
3. 用户点击「AI 视频分割」，选择本地视频并配置切片时长，进入进度页。
4. 分割完成后返回编辑页，自动生成 numbered 分镜头列表。
5. 用户对分镜头进行排序、镜像、替换、删除等操作。
6. 用户添加口播配音和/或背景音乐。
7. 用户点击「AI 生成」，系统校验算力是否充足。
8. 算力充足则开始批量生成，展示生成进度；算力不足则弹窗提示。
9. 生成完成后可预览成片并下载。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 视频组 ID | String | 路由/接口 | 当前视频组唯一标识 |
| 视频组名称 | String | 用户输入/后台 | 可编辑 |
| 分镜头列表 | Array<Object> | 本地/接口 | 编号、视频 URL、排序、是否镜像 |
| 保留原声 | Boolean | 用户选择 | 默认开启或关闭由产品确定 |
| 配音文本 | String | 用户输入 | 文字转语音内容 |
| 配音音频文件 | File/URL | 用户录音/AI 生成 | 可叠加到视频轨道 |
| 背景音乐 | File/URL | 用户上传/提取原视频 | 可叠加到视频轨道 |
| 算力消耗 | Number | 后台配置 | 每生成一条视频消耗的算力 |
| 生成状态 | Enum | 接口返回 | 待生成、生成中、已完成、失败 |

## 5. 异常处理与边界规则

- **算力不足**：AI 生成按钮置灰，点击提示「当前算力已耗尽，请联系管理员增加算力额度」。
- **未上传素材**：AI 生成前校验，若分镜头为空则提示「请先上传或分割视频素材」。
- **分割失败**：提示「视频分割失败，请检查素材后重试」。
- **生成失败**：单条失败展示失败标识，可单独重试。
- **删除分镜头**：二次确认弹窗。

## 6. UI 与视觉规范

- 顶部返回按钮 + 可编辑标题。
- 视频预览区居上，下方为分镜头横向/纵向列表。
- 分镜头卡片带编号、关闭按钮、操作图标。
- AI 视频分割、查看进度按钮蓝色文字 + 淡蓝底色。
- AI 生成按钮为底部固定蓝色主按钮。
- 加载、生成过程需展示进度动画。
