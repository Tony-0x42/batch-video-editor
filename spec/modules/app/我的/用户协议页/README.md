# 用户协议页

## 1. 页面概述

- **页面定位**：「我的」页面中「用户协议」入口的二级页面，展示完整的用户协议文档内容。
- **用户角色**：所有已登录用户。
- **入口来源**：我的 → 用户协议；登录页协议链接。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回上一页。
- **页面标题**：「用户协议」。

### 2.2 文档内容区

- **元素**：富文本/图文混排展示用户协议全文。
- **内容来源**：后台配置更新。
- **操作**：支持上下滑动查看全文；图片支持放大预览。

### 2.3 更新时间

- **元素**：页面底部或标题下方展示文档更新时间。

## 3. 交互流程

1. 用户从「我的」或登录页进入用户协议页。
2. 页面拉取后台配置的用户协议内容。
3. 用户滑动查看完整协议内容。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 协议标题 | String | 后台配置 | 默认「用户协议」 |
| 协议内容 | HTML/String | 后台配置 | 富文本全文 |
| 更新时间 | Date | 后台配置 | 格式 YYYY-MM-DD |

## 5. 异常处理与边界规则

- **后台无协议内容**：展示空状态或默认提示「协议内容加载中」。
- **内容加载失败**：提示「协议加载失败，请重试」。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题。
- 内容区白色背景，正文黑色，标题加粗。
- 段落间距适中，字体大小统一。
