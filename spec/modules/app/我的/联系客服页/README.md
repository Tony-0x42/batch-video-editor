# 联系客服页

## 1. 页面概述

- **页面定位**：「我的」页面中「联系客服」入口的二级页面，展示客服联系方式与服务时段。
- **用户角色**：所有已登录用户。
- **入口来源**：我的 → 联系客服。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回「我的」页面。
- **页面标题**：「联系客服」。

### 2.2 客服信息卡片

- **元素**：卡片展示客服电话、服务时段。
- **展示字段**：
  - 客服电话
  - 服务时段：早 9:00-18:00
- **操作**：点击电话可直接唤起系统拨号。

### 2.3 在线客服入口（可选）

- **元素**：按钮或卡片入口。
- **功能**：点击后跳转在线客服聊天窗口或客服 H5 页面。

### 2.4 常见问题入口（可选）

- **元素**：「常见问题」入口。
- **功能**：点击后跳转新手文档页/FAQ 页面。

## 3. 交互流程

1. 用户从「我的」页面进入联系客服页。
2. 页面拉取后台配置的客服信息。
3. 用户点击电话拨号，或点击在线客服入口咨询。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 客服电话 | String | 后台配置 | 可点击拨号 |
| 服务时段 | String | 后台配置 | 如「早 9:00-18:00」 |
| 在线客服链接 | URL | 后台配置 | 跳转在线客服 |

## 5. 异常处理与边界规则

- **后台无客服配置**：展示空状态「暂无客服信息」。
- **拨号失败**：提示「拨号失败，请检查设备权限」。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题。
- 客服信息卡片蓝色主题或白色卡片。
- 电话文字蓝色可点击。
- 服务时段灰色小字。
