# 数据统计

## 1. 页面概述

- **页面定位**：管理后台中用于汇总和查看平台运营数据、账号数据、算力消耗、视频生成、二维码推广等数据的模块。
- **用户角色**：总后台管理员、分公司管理员（仅查看本分公司下属数据）。
- **入口来源**：后台首页 → 侧边栏「数据统计」。

## 2. 页面元素与功能点

### 2.1 顶部筛选区

- **时间范围选择**：支持按日、周、月、自定义时间范围筛选。
- **账号类型筛选**：分公司 / 服务商 / 个人 / 全部。
- **所属分公司筛选**：总后台可按分公司筛选查看；分公司后台固定为本分公司，不可切换。
- **数据维度切换**：
  - 账号数据
  - 算力数据
  - 视频生成数据
  - 二维码推广数据
  - 业绩喜报数据

### 2.2 数据概览卡片

- **总后台展示指标**：
  - 账号总数（分公司/服务商/个人）
  - 今日新增账号数
  - 今日算力消耗总量
  - 今日视频生成数量
  - 二维码扫码次数、下载量、注册数
  - 喜报业绩总金额
- **分公司后台展示指标**：
  - 本公司服务商总数量
  - 各服务商名下个人账号数量
  - 服务商剩余可创建个人账号名额
  - 分公司剩余可新增服务商名额
  - 本分公司今日新增账号数
  - 本分公司算力消耗总量
  - 本分公司视频生成数量
  - 本分公司二维码扫码、下载、注册统计数据

### 2.3 数据明细表格

- **元素**：根据当前维度展示明细数据。
- **账号数据表字段**：账号类型、账号名称、手机号、上级手机号、创建时间、状态。
- **算力数据表字段**：账号名称、手机号、操作时间、操作类型、消耗算力、剩余算力。
- **视频生成数据表字段**：账号名称、视频组名称、生成时间、生成数量、状态。
- **二维码推广数据表字段**：二维码所属账号、扫码次数、下载量、注册数。
- **业绩喜报表字段**：区域、标题、销售冠军、销售金额、更新时间。

### 2.4 图表数据区

- **元素**：折线图/柱状图/饼图展示数据趋势与占比。
- **展示内容**：
  - 近 7/30 天新增用户趋势
  - 近 7/30 天算力消耗趋势
  - 近 7/30 天视频生成趋势
  - 账号类型分布饼图
  - 二维码推广转化漏斗（可选）

### 2.5 导出功能（可选）

- **元素**：「导出数据」按钮。
- **功能**：将当前筛选条件下的数据导出为 Excel/CSV。

## 3. 交互流程

1. 管理员进入数据统计页面。
2. 页面默认展示今日概览数据。
3. 管理员切换数据维度、筛选时间范围、账号类型、分公司。
4. 页面实时更新数据概览、图表、明细表格。
5. 管理员可查看明细，点击账号名称跳转账号详情页。
6. 管理员可导出数据（如支持）。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 时间范围 | DateRange | 用户选择 | 日/周/月/自定义 |
| 数据概览 | Object | 后台统计 | 各维度汇总数据 |
| 明细数据 | Array<Object> | 后台数据 | 根据维度变化 |
| 趋势数据 | Array<Object> | 后台统计 | 日期 + 数值 |
| 数据权限 | Enum | 登录态 | total / branch |

## 5. 异常处理与边界规则

- **数据加载失败**：展示空状态并提示「数据加载失败，请刷新」。
- **无权限查看分公司数据**：分公司后台无法查看其他分公司任何数据，接口自动隔离。
- **导出失败**：提示「导出失败，请稍后重试」。
- **时间范围非法**：提示「请选择合法的时间范围」。

## 6. UI 与视觉规范

- 顶部为筛选区 + 数据维度 Tab。
- 概览卡片白色背景，关键数字蓝色加粗。
- 图表区高度固定，数据为空时展示空状态。
- 明细表格支持分页、排序。
- 导出按钮位于表格右上角。
