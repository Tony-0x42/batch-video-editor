# 会员 VIP 管理

## 1. 页面概述

- **页面定位**：管理后台中用于集中配置账号 VIP 有效期、VIP 标识规则及批量调整 VIP 信息的模块。
- **用户角色**：总后台管理员。
- **入口来源**：后台首页 → 侧边栏「会员 VIP 管理」。

## 2. 页面元素与功能点

### 2.1 顶部操作区

- **批量调整 VIP 按钮**：点击后弹出批量设置窗口，可批量设置选中账号的 VIP 有效期。
- **搜索框**：支持按手机号、账号名称搜索。
- **筛选条件**：
  - 账号类型：分公司 / 服务商 / 个人 / 全部
  - VIP 状态：有效 / 已过期 / 全部
  - 所属分公司筛选

### 2.2 VIP 账号列表

- **元素**：表格展示所有账号的 VIP 信息。
- **表格字段**：
  - 账号类型
  - 账号名称
  - 联系人
  - 手机号
  - VIP 标识（是/否）
  - VIP 有效期
  - 剩余天数
  - 状态（有效/已过期）
  - 操作
- **操作按钮**：
  - 编辑：修改单个账号的 VIP 有效期
  - 查看详情：跳转账号详情页

### 2.3 单个编辑弹窗

- **元素**：弹窗形式编辑 VIP 有效期。
- **字段**：
  - 账号名称（只读）
  - 手机号（只读）
  - VIP 有效期（日期选择器）
- **操作**：保存、取消。

### 2.4 批量编辑弹窗

- **元素**：弹窗形式批量设置选中账号的 VIP 有效期。
- **字段**：
  - 已选中账号数量
  - VIP 有效期（日期选择器）
- **操作**：保存、取消。

### 2.5 VIP 规则配置（可选）

- **元素**：配置 VIP 标识规则、到期提醒规则等。
- **字段**：
  - 是否展示 VIP 标识
  - 到期前提醒天数
  - 过期后是否禁用功能

## 3. 交互流程

1. 管理员进入会员 VIP 管理页面。
2. 页面加载所有账号的 VIP 信息列表。
3. 管理员搜索/筛选目标账号。
4. 管理员点击「编辑」修改单个账号 VIP 有效期，或勾选多个账号后点击「批量调整」。
5. 保存后实时更新列表数据。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 账号列表 | Array<Object> | 后台数据 | 包含账号基础信息与 VIP 信息 |
| VIP 有效期 | Date | 用户输入 | 格式 YYYY-MM-DD |
| VIP 标识 | Boolean | 后台计算 | 有效期内为 true |
| 剩余天数 | Number | 前端计算 | VIP 有效期 - 当前日期 |
| VIP 规则 | Object | 后台配置 | 标识规则、提醒规则等 |

## 5. 异常处理与边界规则

- **VIP 有效期格式错误**：提示「请选择正确的日期」。
- **批量调整未选中账号**：提示「请至少选择一个账号」。
- **保存失败**：提示「保存失败，请稍后重试」。
- **APP 端展示**：VIP 信息由后台下发，APP 我的页面实时展示。

## 6. UI 与视觉规范

- 页面顶部为筛选区 + 批量操作按钮，下方为表格。
- VIP 有效期过期状态使用红色标记，有效状态使用绿色或蓝色。
- 表格行 hover 效果，操作按钮每行独立展示。
- 弹窗表单简洁，保存按钮蓝色。
