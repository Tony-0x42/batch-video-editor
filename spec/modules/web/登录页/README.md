# 后台登录页

## 1. 页面概述

- **页面定位**：管理后台的登录入口，用于验证管理员身份并进入后台系统。
- **用户角色**：总后台管理员、分公司管理员。
- **入口来源**：浏览器访问后台域名后默认进入。

## 2. 页面元素与功能点

### 2.1 品牌展示区

- **元素**：后台系统 Logo、产品名称「批量剪辑 管理后台」。
- **数据约束**：Logo 与名称由后台配置，前端只读展示。

### 2.2 登录表单

- **账号输入框**：
  - 占位文案：请输入管理员账号/手机号
  - 支持手机号或账号名登录
- **密码输入框**：
  - 占位文案：请输入密码
  - 支持显示/隐藏密码切换
- **验证码输入框**（可选）：
  - 占位文案：请输入验证码
  - 右侧展示图形验证码图片，点击可刷新

### 2.3 登录按钮

- **元素**：蓝色主按钮，文案「登录」。
- **状态**：表单未填写完整时置灰，填写完整后高亮可点击。
- **点击后**：
  - 前端校验账号密码格式
  - 调用登录接口
  - 登录成功后跳转后台首页
  - 登录失败时提示错误信息

### 2.4 记住密码（可选）

- **元素**：复选框「记住密码」。
- **功能**：勾选后本地记住账号密码，下次自动填充。

## 3. 交互流程

1. 管理员访问后台域名，进入登录页。
2. 输入账号、密码、验证码（如有）。
3. 点击「登录」，前端校验必填项。
4. 调用登录接口，成功后缓存 Token 并跳转后台首页。
5. 失败时根据错误码提示：账号不存在、密码错误、验证码错误、账号被禁用等。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 账号/手机号 | String | 用户输入 | 管理员登录账号 |
| 密码 | String | 用户输入 | 加密后提交 |
| 图形验证码 | String | 用户输入 | 图形验证码校验 |
| Token | String | 登录接口返回 | 用于后续接口鉴权 |
| 管理员角色 | Enum | 登录接口返回 | total（总后台）/ branch（分公司） |

## 5. 异常处理与边界规则

- **账号或密码为空**：提示「请输入账号和密码」。
- **验证码错误**：提示「验证码错误，请重新输入」并刷新验证码。
- **账号或密码错误**：提示「账号或密码错误」。
- **账号被禁用**：提示「账号已被禁用，请联系总后台管理员」。
- **非管理员账号**：提示「该账号无后台登录权限」。
- **网络异常**：提示「网络连接失败，请检查网络后重试」。

## 6. UI 与视觉规范

- 页面居中登录卡片，宽度固定（如 400px），白色背景 + 圆角 + 阴影。
- 背景建议使用浅蓝色/灰色渐变或品牌相关图片。
- 输入框圆角边框，聚焦时蓝色边框。
- 登录按钮蓝色填充、全宽、圆角。
- 页面整体风格简洁商务，主色与 APP 保持一致（蓝色）。
