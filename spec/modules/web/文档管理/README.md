# 文档管理

## 1. 页面概述

- **页面定位**：管理后台中用于管理 APP 端各类文档内容的模块，包括新手文档、用户协议、隐私政策、帮助文档等。
- **用户角色**：总后台管理员。
- **入口来源**：后台首页 → 侧边栏「文档管理」。

## 2. 页面元素与功能点

### 2.1 顶部操作区

- **新增文档按钮**：蓝色主按钮，点击弹出新增文档弹窗/跳转新增页。
- **搜索框**：支持按文档标题搜索。
- **筛选条件**：
  - 文档类型：用户协议 / 隐私政策 / 新手文档 / 帮助文档 / 全部
  - 状态：已启用 / 已禁用 / 全部

### 2.2 文档列表

- **元素**：表格展示所有文档。
- **表格字段**：
  - 文档标题
  - 文档类型
  - 适用页面（如登录页、我的页、新手文档页）
  - 更新时间
  - 状态（启用/禁用）
  - 操作
- **操作按钮**：
  - 查看：预览文档内容
  - 编辑：修改文档内容
  - 启用/禁用：切换状态
  - 删除：删除文档（二次确认，系统默认文档不可删除）

### 2.3 新增/编辑文档

- **元素**：弹窗或独立页面。
- **字段**：
  - 文档标题（必填）
  - 文档类型（单选）
  - 适用页面（多选/单选）
  - 文档内容（富文本编辑器，必填）
  - 排序权重（可选）
  - 状态（启用/禁用）

### 2.4 文档预览

- **元素**：弹窗展示文档在 APP 端的展示效果。
- **展示内容**：标题、更新时间、正文。

## 3. 交互流程

1. 管理员进入文档管理页面。
2. 页面加载文档列表。
3. 管理员新增文档，选择类型、填写标题与内容。
4. 管理员可编辑、启用/禁用、删除文档。
5. 启用状态的文档在 APP 端对应页面展示。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 文档标题 | String | 用户输入 | 必填 |
| 文档类型 | Enum | 用户选择 | user_agreement / privacy_policy / guide / help |
| 适用页面 | Array<String> | 用户选择 | 登录页、我的页、新手文档页等 |
| 文档内容 | HTML/String | 用户输入 | 富文本，必填 |
| 排序权重 | Number | 用户输入 | 影响同一类型文档展示顺序 |
| 状态 | Enum | 用户选择 | 启用/禁用 |
| 更新时间 | Date | 系统自动 | 保存时更新 |

## 5. 异常处理与边界规则

- **标题或内容为空**：提示「请填写文档标题和内容」。
- **同类型文档重复**：同一适用页面同一类型最多启用一份，冲突时提示「该位置已存在启用的文档，请先禁用旧文档」。
- **系统默认文档不可删除**：如系统默认用户协议、隐私政策，删除按钮置灰。
- **保存失败**：提示「保存失败，请稍后重试」。

## 6. UI 与视觉规范

- 顶部新增按钮 + 筛选区 + 搜索框。
- 文档列表表格展示，文档类型使用标签区分。
- 富文本编辑器使用统一组件。
- 预览弹窗模拟 APP 端文档展示效果。
