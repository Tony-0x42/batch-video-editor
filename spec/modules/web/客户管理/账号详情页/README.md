# 账号详情页

## 1. 页面概述

- **页面定位**：客户管理模块下的二级页面，展示单个账号的完整信息、下级账号、算力消耗、视频生成记录等。
- **用户角色**：总后台管理员、分公司管理员（仅查看本分公司下属账号）。
- **入口来源**：客户管理 → 列表「查看」按钮。

## 2. 页面元素与功能点

### 2.1 顶部导航栏

- **返回按钮**：点击返回客户管理列表页。
- **页面标题**：「账号详情」。
- **编辑按钮**：总后台管理员可点击编辑账号信息。

### 2.2 基础信息卡片

- **展示字段**：
  - 账号类型
  - 账号名称
  - 联系人
  - 手机号
  - 上级手机号
  - 所属分公司
  - 状态（启用/禁用）
  - 创建时间
  - VIP 有效期
  - 算力总配额 / 已消耗算力 / 剩余算力

### 2.3 配额信息

- **分公司账号展示**：
  - 最大可创建服务商数量 X
  - 已创建服务商数量
  - 剩余可创建服务商名额
  - 旗下服务商总可分配个人账号容量 Y
  - 已分配个人账号数量
  - 剩余可分配名额
- **服务商账号展示**：
  - 可拆分创建个人账号上限 X
  - 已创建个人账号数量
  - 剩余可创建名额

### 2.4 下级账号列表

- **元素**：表格展示当前账号的直接下级账号。
- **表格字段**：账号类型、账号名称、联系人、手机号、创建时间、状态、操作。
- **操作**：查看详情、迁移、删除（总后台权限）。

### 2.5 算力消耗记录

- **元素**：列表/图表展示该账号的算力消耗明细。
- **字段**：操作时间、操作类型（生成/下载）、消耗算力、关联视频组。

### 2.6 视频生成记录

- **元素**：列表展示该账号的视频生成记录。
- **字段**：生成时间、视频组名称、生成数量、状态（成功/失败）。

### 2.7 注册二维码

- **元素**：展示该账号的专属注册二维码。
- **操作**：下载二维码图片、复制推广链接、重置二维码。

### 2.8 操作按钮

- **编辑**：跳转编辑模式。
- **升级**：个人→服务商、服务商→分公司（总后台权限）。
- **迁移**：修改上级手机号（总后台权限）。
- **禁用/启用**：切换账号状态。
- **删除**：删除账号（存在下级时拦截）。

## 3. 交互流程

1. 管理员从客户管理列表点击「查看」进入账号详情页。
2. 页面加载账号基础信息、配额、下级列表、算力记录、视频记录、二维码。
3. 管理员可切换不同 Tab 查看不同维度的数据。
4. 管理员点击操作按钮进行编辑、升级、迁移、禁用、删除等操作。
5. 高危操作需二次确认。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 账号基础信息 | Object | 后台数据 | 类型、名称、联系人、手机号等 |
| 配额信息 | Object | 后台计算 | 上下限与已用数量 |
| 下级账号列表 | Array<Object> | 后台数据 | 当前账号的直接下级 |
| 算力消耗记录 | Array<Object> | 后台数据 | 时间、类型、消耗值 |
| 视频生成记录 | Array<Object> | 后台数据 | 生成记录明细 |
| 注册二维码 | URL | 后台生成 | 含上级手机号参数 |

## 5. 异常处理与边界规则

- **账号不存在**：提示「账号不存在或已被删除」，返回列表页。
- **无权限查看**：提示「无权限查看该账号信息」。
- **删除存在下级**：拦截并提示「该账号存在下级账号，请先迁移或清空下级」。
- **二维码重置**：二次确认后重置，旧二维码立即失效。

## 6. UI 与视觉规范

- 顶部返回按钮 + 标题 + 编辑按钮。
- 基础信息使用卡片布局，关键数据蓝色高亮。
- 下级列表、算力记录、视频记录使用 Tab 切换。
- 二维码区域居中展示，下方有下载、复制、重置按钮。
- 高危操作按钮红色，主操作按钮蓝色。
