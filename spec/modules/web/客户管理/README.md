# 客户管理

## 1. 页面概述

- **页面定位**：总后台管理员管理全平台账号的核心模块，支持新增、编辑、删除、查询分公司/服务商/个人账号，以及查看账号详情和注册二维码。
- **用户角色**：总后台管理员、分公司管理员（仅查看本分公司下属账号）。
- **入口来源**：后台首页 → 侧边栏「客户管理」。
- **权限说明**：仅总后台拥有新增账号、编辑账号、删除账号、调整配额、升级账号等级、迁移账号层级权限；分公司后台仅展示本分公司下属账号数据，无新增、编辑、删除、升级、迁移等操作权限。

## 2. 页面元素与功能点

### 2.1 顶部操作区

- **新增账号按钮**：蓝色主按钮，点击跳转「新增账号页」。
- **搜索框**：支持按手机号、账号名称、联系人模糊搜索。
- **筛选条件**：
  - 账号类型筛选：分公司 / 服务商 / 个人 / 全部
  - 所属分公司筛选（总后台可见）
  - 状态筛选：启用 / 禁用 / 全部

### 2.2 账号列表

- **元素**：表格展示所有账号。
- **表格字段**：
  - 账号类型（分公司/服务商/个人）
  - 账号名称
  - 联系人
  - 手机号
  - 上级手机号（服务商/个人展示）
  - 所属分公司（服务商/个人展示）
  - 算力总配额（GF）
  - VIP 有效期
  - 下级数量（分公司/服务商展示）
  - 状态（启用/禁用）
  - 创建时间
  - 操作
- **操作按钮**（总后台权限）：
  - 查看：进入账号详情页
  - 编辑：进入编辑模式
  - 二维码：查看注册二维码
  - 删除：删除账号（存在下级时拦截）
  - 升级：个人→服务商、服务商→分公司
  - 迁移：修改上级手机号
- **分公司后台操作按钮**：
  - 查看：进入账号详情页（仅查看本分公司下属账号，且数据范围受限）

### 2.3 分页器

- **元素**：表格底部分页器。
- **功能**：支持页码切换、每页条数选择（如 10/20/50）。

### 2.4 空状态

- **触发条件**：无账号数据或搜索无结果。
- **展示**：空状态占位图 +「暂无账号数据」文案。

## 3. 交互流程

1. 管理员进入客户管理页面。
2. 页面默认加载全部账号列表。
3. 管理员可使用搜索、筛选查找目标账号。
4. 管理员点击「新增账号」进入新增流程。
5. 管理员点击列表操作按钮，进行查看、编辑、删除、升级、迁移、查看二维码等操作。
6. 删除账号时，系统校验是否存在下级账号，存在则弹窗拦截。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 账号 ID | String | 后台数据 | 唯一标识 |
| 账号类型 | Enum | 后台数据 | branch_company / service_provider / individual |
| 账号名称 | String | 用户输入/后台 | 创建时填写 |
| 联系人 | String | 用户输入/后台 | 创建时填写 |
| 手机号 | String | 用户输入/后台 | 全局唯一 |
| 上级手机号 | String | 后台数据 | 绑定关系标识 |
| 算力总配额 | Number | 后台配置 | 单位 GF |
| VIP 有效期 | Date | 后台配置 | 格式 YYYY-MM-DD |
| 状态 | Enum | 后台数据 | enabled / disabled |
| 下级数量 | Number | 后台计算 | 当前账号下级账号数 |

## 5. 异常处理与边界规则

- **手机号重复**：新增/编辑时校验，重复则弹窗拦截并提示「该手机号已被注册」。
- **上级手机号不存在/类型不匹配**：弹窗拦截并提示「上级手机号不存在或类型不匹配」。
- **下级名额已满**：弹窗拦截并提示「上级可创建下级名额已满」。
- **账号存在下级**：删除时拦截，提示「该账号存在下级账号，请先迁移或清空下级」。
- **升级后层级冲突**：升级时校验手机号是否已归属其他类型，冲突则拦截。

## 6. UI 与视觉规范

- 页面顶部为筛选区 + 新增按钮，下方为表格。
- 表格行 hover 效果，操作按钮每行独立展示或收起在「更多」下拉中。
- 高危操作按钮（删除）使用红色。
- 主操作按钮（新增、搜索）使用蓝色。
- 分页器居右，页码选中项蓝色。
