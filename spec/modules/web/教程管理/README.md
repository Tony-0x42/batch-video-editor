# 教程管理

## 1. 页面概述

- **页面定位**：管理后台中用于上传、编辑、下架 APP 学习专区教学视频和图文教程的模块。
- **用户角色**：总后台管理员。
- **入口来源**：后台首页 → 侧边栏「教程管理」。

## 2. 页面元素与功能点

### 2.1 顶部操作区

- **新增教程按钮**：蓝色主按钮，点击弹出新增教程弹窗/跳转新增页。
- **搜索框**：支持按教程标题搜索。
- **筛选条件**：
  - 教程类型：视频 / 文档 / 全部
  - 分类
  - 状态：已上架 / 已下架 / 全部

### 2.2 教程列表

- **元素**：表格展示所有教程。
- **表格字段**：
  - 教程标题
  - 教程类型（视频/文档）
  - 分类
  - 封面图缩略图
  - 发布时间
  - 状态（已上架/已下架）
  - 浏览次数（可选）
  - 操作
- **操作按钮**：
  - 查看：预览教程
  - 编辑：修改教程信息
  - 上架/下架：切换状态
  - 删除：删除教程（二次确认）

### 2.3 新增/编辑教程

- **元素**：弹窗或独立页面。
- **字段**：
  - 教程标题（必填）
  - 教程类型（视频/文档，单选）
  - 分类选择（从分类管理中选取）
  - 封面图上传（必填）
  - 视频文件上传 / 文档内容编辑（根据类型）
  - 简介（可选）
  - 排序权重
  - 状态（上架/下架）

### 2.4 教程分类管理（可选）

- **元素**：管理教程分类。
- **功能**：新增、编辑、删除分类，调整分类排序。

## 3. 交互流程

1. 管理员进入教程管理页面。
2. 页面加载教程列表与分类。
3. 管理员新增教程，选择类型、分类、上传封面与内容。
4. 管理员可编辑、上架、下架、删除教程。
5. 已上架教程在 APP 学习专区展示。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 教程标题 | String | 用户输入 | 必填 |
| 教程类型 | Enum | 用户选择 | video / document |
| 分类 | String/Object | 用户选择/后台配置 | 学习专区分类 |
| 封面图 | URL | 用户上传 | 必填 |
| 视频文件/文档内容 | File/HTML | 用户上传/输入 | 根据类型必填 |
| 简介 | String | 用户输入 | 可选 |
| 排序权重 | Number | 用户输入 | 影响展示顺序 |
| 状态 | Enum | 用户选择 | 上架/下架 |
| 浏览次数 | Number | 后台统计 | 可选 |

## 5. 异常处理与边界规则

- **必填项为空**：提示「请完善教程信息」。
- **文件上传失败**：提示「上传失败，请检查文件格式和大小后重试」。
- **视频格式不支持**：提示「请上传 MP4 格式视频，大小不超过 XXX MB」。
- **删除教程**：二次确认后删除。
- **分类被使用**：删除分类时若存在关联教程，提示「该分类下存在教程，无法删除」。

## 6. UI 与视觉规范

- 顶部新增按钮 + 筛选区 + 搜索框。
- 教程列表表格展示，封面图使用缩略图。
- 视频/文档类型使用不同图标或标签颜色区分。
- 编辑弹窗分步骤或分区块展示基础信息和内容上传。
