# 首页内容管理

## 1. 页面概述

- **页面定位**：管理后台中用于配置 APP 首页展示内容的模块，包括轮播图、喜报数据、功能入口、教程入口等。
- **用户角色**：总后台管理员。
- **入口来源**：后台首页 → 侧边栏「首页内容管理」。

## 2. 页面元素与功能点

### 2.1 Tab 切换

- **元素**：顶部 Tab，切换不同内容类型。
- **Tab 项**：
  - 轮播图管理
  - 喜报数据管理
  - 功能入口配置
  - 教程入口配置

### 2.2 轮播图管理

- **列表字段**：
  - 排序权重
  - 图片缩略图
  - 标题（可选）
  - 跳转链接（可选）
  - 状态（启用/禁用）
  - 操作（编辑、删除、上下移动）
- **新增/编辑弹窗**：
  - 图片上传（必填）
  - 标题输入
  - 跳转链接输入
  - 排序权重
  - 状态开关

### 2.3 喜报数据管理

- **展示字段**：
  - 业绩标题
  - 销售冠军姓名
  - 销售金额
  - 状态（启用/禁用）
  - 更新时间
  - 操作（编辑、删除）
- **新增/编辑弹窗**：
  - 业绩标题输入
  - 销售冠军姓名输入
  - 销售金额输入
  - 状态开关

### 2.4 功能入口配置

- **展示字段**：
  - 入口名称
  - 入口图标
  - 跳转目标
  - 排序
  - 状态（启用/禁用）
  - 操作（编辑、删除）
- **新增/编辑弹窗**：
  - 入口名称
  - 图标上传
  - 跳转目标选择/输入
  - 排序权重
  - 状态开关
- **说明**：「其他服务」入口可配置为预留空白状态。

### 2.5 教程入口配置

- **展示字段**：
  - 入口标题
  - 封面图
  - 关联文档
  - 排序
  - 状态
  - 操作
- **新增/编辑弹窗**：
  - 入口标题
  - 封面图上传
  - 选择关联文档（从文档管理中选择）
  - 排序权重
  - 状态开关

## 3. 交互流程

1. 管理员进入首页内容管理页面。
2. 切换 Tab 选择要管理的内容类型。
3. 管理员可新增、编辑、删除、排序、启用/禁用各类内容。
4. 保存后 APP 端首页实时拉取最新配置展示。

## 4. 数据字段与来源

| 字段 | 类型 | 来源 | 说明 |
|---|---|---|---|
| 轮播图列表 | Array<Object> | 后台配置 | 图片、标题、链接、排序、状态 |
| 喜报数据 | Object | 后台配置 | 标题、姓名、金额、状态 |
| 功能入口列表 | Array<Object> | 后台配置 | 名称、图标、跳转目标、排序、状态 |
| 教程入口列表 | Array<Object> | 后台配置 | 标题、封面、关联文档、排序、状态 |

## 5. 异常处理与边界规则

- **图片上传失败**：提示「图片上传失败，请重试」。
- **必填项为空**：提示「请完善信息」。
- **图片格式/大小不符**：提示具体限制，如「请上传 JPG/PNG 格式，大小不超过 5MB」。
- **APP 端无配置**：展示默认空白占位图，不报错崩溃。
- **删除二次确认**：删除轮播图、喜报等时二次确认。

## 6. UI 与视觉规范

- 顶部 Tab 选中项蓝色下划线。
- 列表以卡片或表格形式展示，图片带缩略图。
- 新增/编辑使用弹窗或抽屉形式。
- 排序支持拖拽或上下箭头。
- 状态开关使用统一组件，开启为蓝色。
