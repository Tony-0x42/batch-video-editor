# 项目边界

> 明确 VideoEditor-For-Android 商用演进版本「做什么」与「不做什么」，以及首个 MVP 的精确范围。

---

## 1. 项目目标

将现有 Demo 级视频编辑器升级为**可在 Android 手机上独立运行的商用短视频剪辑应用**，核心交付物是一个安装包（APK），用户无需后端服务即可完成基础剪辑与导出。

---

## 2. 项目边界（In Scope）

### 2.1 必须实现的核心能力

| 类别 | 功能 |
|------|------|
| **素材导入** | 从相册导入视频、图片、音频；支持多选；扫描媒体库 |
| **时间线编辑** | 多轨道时间轴；主视频轨 + 画中画轨 + 音频轨 + 文字轨 |
| **基础剪辑** | 分割、裁剪头尾、删除、复制、拖拽排序、撤销/重做 |
| **画布与比例** | 9:16 / 16:9 / 1:1 / 4:3；背景模糊/纯色/图片填充 |
| **音频处理** | 本地 BGM、音频提取、音量调节、淡入淡出、基础混音 |
| **文字字幕** | 手动添加文字、字体/颜色/描边/阴影、位置与旋转、出入场动画 |
| **滤镜美颜** | 20+ 预设滤镜、滤镜强度调节、美颜/磨皮/美白 |
| **导出分享** | 720p/1080p、30fps、MP4、码率预设、保存相册、导出进度 |
| **草稿管理** | 自动保存编辑状态、草稿列表、重命名/删除 |
| **水印控制** | 导出时可开关水印 |

### 2.2 保留并增强的现有能力

- 基于 OpenGL 的滤镜链
- MediaCodec 硬编码导出
- 相机录制（迁移到 Camera2/CameraX）
- JNI 音频混音

### 2.3 支持的输入格式

| 类型 | 格式 |
|------|------|
| 视频 | MP4 (H.264/H.265)、MOV |
| 图片 | JPG、PNG、WEBP |
| 音频 | MP3、AAC、WAV |

### 2.4 支持的输出格式

| 类型 | 格式 |
|------|------|
| 视频 | MP4 (H.264/AAC) |
| 分辨率 | 480p / 720p / 1080p |
| 帧率 | 24fps / 30fps |

## 3. 明确不做（Out of Scope）

### 3.1 第一阶段不做

| 类别 | 功能 | 原因 |
|------|------|------|
| **云端服务** | 云同步、云草稿、在线素材库、用户账号系统 | 需要后端与基础设施，超出当前范围 |
| **AI 能力** | 智能字幕、AI 配音、智能抠像、AI 成片 | 依赖云端 AI 或大型模型，后续迭代 |
| **高级调色** | LUT 导入、HSL、RGB 曲线、色轮 | 用户门槛高，优先基础调色 |
| **复杂特效** | 3D 特效、粒子系统、复杂转场 | 资源投入大，后续迭代 |
| **直播/拍摄** | 直播推流、高清录屏、提词器 | 非剪辑核心能力 |
| **多平台** | iOS、PC、Web 版本 | 聚焦 Android 单平台 |
| **商业化系统** | 会员体系、支付、广告、素材商店 | 产品验证后再接入 |

### 3.2 第二阶段待评估

以下功能不在首个 MVP，但架构上需预留扩展点：
- 变速、倒放、定格
- 关键帧动画
- 转场效果库
- 音效库与录音
- 贴纸库
- 画中画增强

## 4. MVP 范围（首个可发布版本）

### 4.1 MVP 必须完成

1. **首页与草稿箱**
   - 开始创作入口
   - 草稿列表（继续编辑、删除、重命名）

2. **素材选择页**
   - 从相册多选视频/图片/音频
   - 最近项目快速进入

3. **剪辑主页面**
   - 顶部播放器实时预览
   - 底部多轨道时间轴
   - 工具栏：分割、删除、复制、撤销/重做
   - 比例与背景设置

4. **功能面板**
   - 音频面板：BGM、音量、淡入淡出、提取音频
   - 文字面板：添加文字、样式、动画
   - 滤镜面板：滤镜选择 + 强度
   - 美颜面板：磨皮、美白

5. **导出页**
   - 分辨率/帧率/码率选择
   - 水印开关
   - 导出进度与取消
   - 导出完成后保存到相册

### 4.2 MVP 不做

- 画中画多层级编辑（保留架构扩展点）
- 曲线变速、倒放、定格
- 自动字幕识别
- 在线素材库
- 社交分享 SDK（仅保存相册）
- 会员与付费

## 5. 用户场景

### 5.1 核心用户故事

> 作为短视频创作者，我希望在手机上快速剪辑一段带 BGM 和字幕的视频，并能一键导出到相册，以便发布到社交平台。

### 5.2 典型流程

1. 打开 App → 点击「开始创作」
2. 从相册选择 3-5 段视频
3. 进入剪辑页，按顺序排列片段
4. 删除多余部分，分割保留精彩片段
5. 添加 BGM，调整音量
6. 添加片头文字和滤镜
7. 选择 9:16 比例，设置模糊背景
8. 导出 1080p 视频到相册

---

## 6. 成功标准

### 6.1 功能标准

- MVP 必须覆盖上述 5 个核心用户故事步骤
- 导出 1 分钟 1080p 视频在中档设备上 ≤ 45 秒
- 崩溃率 < 1%（MVP 阶段可放宽）

### 6.2 体验标准

- 新手用户无需教程即可完成「导入 → 剪辑 → 导出」
- 时间线操作响应流畅，无明显卡顿
- UI 风格统一，符合现代 Android 设计规范
