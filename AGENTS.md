# 批量剪辑 - Agent 工作指南

本文件汇总项目级通用信息，供开发 Agent 在编写、修改、 review 代码前快速了解项目背景与约束。详细页面级需求请查阅 `spec/` 目录下对应模块文档。

## 1. 项目简介

| 项 | 说明 |
|---|---|
| 产品名称 | 批量剪辑 |
| 产品定位 | 面向品牌方、经销商及个人用户的 AI 短视频创作与去水印工具型 APP，配套管理后台实现内容、账号、算力、层级的统一管理 |
| 产品形态 | APP（用户端）+ Web（管理后台） |

## 2. 产品形态

| 端 | 使用对象 | 说明 |
|---|---|---|
| APP | 分公司、服务商、个人用户 | 核心功能为 AI 视频创作、去水印、学习资料查看、个人中心 |
| Web 管理后台 | 总后台管理员、分公司管理员 | 用于内容配置、账号管理、数据统计、系统配置 |

## 3. 核心功能模块

### 3.1 APP 端

| 模块 | 核心功能 |

| 登录页 | 手机号 + 密码登录 |
| 注册页 | 扫码后读取上级手机号，输入手机号和密码完成注册 |

| 我的 | 账号信息、VIP 信息、客服、账号绑定、系统设置、退出登录 |

## 4. 全局约束

### 4.1 视觉与交互

- **主色调**：蓝色；所有主操作按钮统一为蓝色填充样式。
- **APP 底部导航**：一级页面底部固定 4 项导航，顺序固定为「首页 / AI 云创 / AI 去水印 / 我的」，当前选中项高亮。
- **返回按钮**：所有二级页面顶部标配返回按钮。
- **加载与空状态**：素材加载、视频解析、切片生成等异步操作必须提供加载动画与空状态占位。
- **高危操作**：删除、注销、解绑等必须增加二次确认弹窗。

### 4.2 数据约束

- **后台下发**：所有前端展示类文本、图片、数字、联系方式、文档、视频素材均由后台下发，前端只读展示，无本地编辑入口。
- **算力全局规则**：
  - 算力为账号维度独立数据，不同账号互不通用。
  - 算力耗尽时禁用视频生成、下载功能，并统一弹窗提示「当前算力已耗尽，请联系管理员增加算力额度」。
- **上传限制**：AI 云创相册单次选择视频上限 10 个；AI 分割切片时长区间固定 0.5~10s，步长 0.1s，前端不可修改区间阈值。

## 5. 组织树与账号层级

- **三级组织树**：分公司 → 服务商 → 个人。
- **上下级唯一关联标识**：手机号。
- **手机号全局唯一**：一个手机号仅能归属一类账号（分公司 / 服务商 / 个人三选一），不可重复注册。
- **账号升级**：总后台可将现有手机号对应账号升级（个人→服务商、服务商→分公司），升级后自动变更层级归属，同步更新所有下级账号的上级关联手机号。
- **账号删除前置校验**：若存在下级账号，系统拦截删除，必须先迁移/清空全部下级。
- **权限范围**：
  - 仅总后台可新增账号、调整配额、升级账号等级、迁移账号层级、重置二维码。
  - 分公司后台仅查看本分公司下属数据，无新增/编辑账号、调整配额、升级账号、迁移层级权限。
  - 服务商无后台登录入口，仅使用 APP。

## 7. Agent 开发约定

- **优先读 spec**：修改或实现任何页面前，先阅读 `spec/modules/` 下对应端与页面的 `README.md`。
- **以手机号为核心的数据模型**：所有层级、上下级、注册二维码绑定均以手机号作为唯一凭证。
- **权限先行**：管理后台功能实现前，先明确当前用户是「总后台管理员」还是「分公司管理员」，再决定数据范围与操作按钮显隐。
- **算力检查**：涉及视频生成、下载的入口必须在交互与 UI 层面前置检查算力余额。
- **素材只读**：APP 端所有展示素材均由后台接口下发，禁止在客户端写死文案、图片、数字、视频等内容。
- **蓝色主色 + 二次确认**：保持视觉一致性，高危操作必须二次确认。
- **最小改动**：做最小必要修改，避免在单 PR/任务中同时改动无关模块。
